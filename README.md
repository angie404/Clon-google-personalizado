# Clon-google-personalizado
Primer proyecto de clonacion de google como parte de los ejercicios en el curso de Tecnolochicas


¡Bienvenido/a al clon de Google personalizado con Pompurín, creado como proyecto de Tecnolóchicas! Esta página web es una recreación del famoso motor de búsqueda de Google, con personalización agregada basada en los gustos personales del autor.

## Acerca del Proyecto

Este proyecto consiste en un clon de la página de inicio de Google, desarrollado desde cero utilizando HTML y CSS. Además de replicar la funcionalidad básica de búsqueda de Google, se ha agregado un toque personalizado con Pompurín en la interfaz gráfica.

## Funcionalidades

- Barra de búsqueda que imita el comportamiento de Google.
- Iconos en el head que simulan los de la interfaz de Google.

## Tecnologías Utilizadas

Este proyecto fue creado utilizando las siguientes tecnologías:

- HTML: Para la estructura y contenido de la página.
- CSS: Para el diseño y estilo visual.


## Instalación y Uso

1. Clona este repositorio en tu máquina local.
2. Abre el archivo index.html en tu navegador web.
3. Explora las diferentes funcionalidades y disfruta la personalización de Pompurín.
